Principais Formatações

Negrito/Destaque

Uma frase contendo uma parte em negrito, usando a tag <strong></strong>

Ítalico/Ênfase

Uma frase contendo uma parte em itálico/ênfase, usando a tag <em></em>

Texto Marcado/Grifado

Uma frase contendo uma parte marcada/grifada, usando a tag <mark></mark>

Texto Grande e Pequeno

Para inserir um texto grande, usamos a tag <big></big> e para um texto pequeno, usamos a tag <small></small>

Texto Deletado

Um texto pode ser marcado como excluído, indicando que ele deve ser lido, mas não considerado. Para tal usamos a tag <del></del>

Texto Inserido

Um texto pode ser dito como inserido, dando ênfase e indicando que foi adicionado depois. Para tal usamos a tag <ins></ins>

Texto Sobrescrito

Para adicionar um texto que possua sobrescrições como x20+x13+3=0, usamos a tag <sup></sup>

Texto Subscrito

Para adicionar um texto que possua subscrições como (C3H6)n, usamos a tag <sub></sub>
