Outras Formatações

Código-fonte/Pré-formatação

É possível inserir códigos de forma que as letras fiquem monoespaçadas assim como em abientes de programação. Para tal usamos a tag <code></code>. Desta forma podemos inserir um trecho de código como: 

        
            num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

Mas note que neste caso acima, não somente foi utilizado a tag <code>, também foi utilizada a tag <pre></pre> que faz com que o texto dentro dela se mantenha pré-formatado, de forma que fique sem alteração de como foi digitado inicialmente.

Citações Completas

Além das citações simples, é possível gerar blocos separados de citações mais longas usando a tag <blockquote></blockquote>:

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Quod numquam repudiandae reiciendis quo sunt eveniet libero consequuntur facere omnis error nesciunt, voluptatibus distinctio! Deleniti officia culpa accusantium voluptatum sequi placeat!

Sem contar que ainda é possível adicionar o link que direciona ao texto original utilizando cite="" após <blockquote.

Abreviações

Ao utilizar abreviações nos textos, podemos adicionar seus significados às mesmas ao usar a tag <abbr title=""></abbr>. Assim, siglas como HTML e CSS continuam simples, mas contendo o significado completo.

Texto Invertido

A tag <bdo dir=""></bdo> pode inverter o texto em seu interior de forma que a sua direção seja especificada como rtl ou ltr. Barbariza, mona!
